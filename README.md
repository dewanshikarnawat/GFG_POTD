## 👨‍💻 About This Repository

This repository contains my **daily solutions** to the [GeeksForGeeks Problem of the Day (POTD)](https://www.geeksforgeeks.org/problem-of-the-day) challenge — solved in **C++** with a focus on:

- ✅ **Correctness** — All solutions are accepted on the GFG platform
- ⚡ **Efficiency** — Optimized time & space complexity
- 📖 **Readability** — Clean, well-structured code
- 🔁 **Consistency** — Committed to solving every single day

GFG_POTD/

## 🚀 How to Use

### Clone the repository

```bash
git clone https://github.com/dewanshikarnawat/GFG_POTD.git
cd GFG_POTD
```

### Compile and run any solution

```bash
g++ -O2 -o solution Medium_Problems/<filename>.cpp
./solution
```

> Requires GCC / G++ (C++17 recommended)
